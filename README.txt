# LSTM-XGBoost prediction of box office revenue

## From Kjell et al 2023 The text-package 
Further, it has been demonstrated that later BERT layers are more context specific (Ethayarajh, 2019)

### Dataset
All information regarding the movies was downloaded from (IMDb){https://developer.imdb.com/non-commercial-datasets/} on 12/April/2024. No script used, downloaded manually.

As the IMDb box-office data is restricted behind a paywall, I used Tom 'tjwaterman99' Waterman (Github repository){https://github.com/tjwaterman99/boxofficemojo-scraper.git}.
The scrape from 12/04/2024 can be found in the data-folder under path: './Data/revenues_per_day.csv'
